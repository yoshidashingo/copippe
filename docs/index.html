copippe

A full-featured clipboard manager for macOS. History, snippets, search, and global hotkeys.

Download User Guide GitHub

English · 日本語

Clipboard

  • Strips rich text formatting on copy — always paste plain text
  • Up to 30 text and image clipboard history entries, configurable in Preferences
  • Copied images saved as PNG thumbnails, restorable to clipboard
  • History persists across restarts

Snippets

  • Register and manage frequently used text snippets
  • Organize snippets into folders for quick access
  • Paste snippets from menu bar folder submenus
  • Assign individual global hotkeys to paste specific snippets instantly

Search & Quick Access

  • Press Shift+Cmd+V from any app to open the floating search popup
  • Incremental search across history and snippets with keyboard navigation
  • Tab switching between history and snippet views

General

  • Runs in the menu bar with no Dock icon
  • Preferences window for hotkeys, history limit, startup, and snippet management
  • Auto-launches at login
  • Built with Swift & SwiftUI — minimal memory footprint

Installation

Download from GitHub Releases, unzip, move to Applications, and launch.

Requires macOS 14 (Sonoma) or later.

This app is not notarized. macOS will block it on first launch.

Run in Terminal: xattr -cr /Applications/copippe.app

Or: try to open the app → System Settings → Privacy & Security → Open Anyway

FAQ

Is it safe to use?
Yes. The source code is fully available on GitHub. You can review and build it yourself.
Can I get support?
You can reach us at support@copippe.com. However, bug reports and feature requests are only accepted as pull requests on GitHub.
Why is it distributed outside the App Store?
Global hotkey support requires permissions not available under App Sandbox. copippe is distributed directly via GitHub Releases.
