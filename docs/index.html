copippe

A simple, lightweight clipboard tool for macOS. Strip formatting, keep history, all from your menu bar.

Download GitHub

Features

✂️

Plain Text Paste

Automatically strips rich text formatting — HTML, RTF, styles — on copy. Always paste clean, plain text.

📜

Clipboard History

Access up to 20 recent clipboard entries from the menu bar. Click to copy back, paste with ⌘V.

🪶

Lightweight

Built with Swift and SwiftUI. Minimal memory footprint, no Dock icon, stays out of your way.

💾

Persistent History

Clipboard history is saved locally and restored across app restarts.

🚀

Auto Launch

Starts automatically at login via macOS Login Items. Set it and forget it.

🔒

Privacy First

Everything stays on your Mac. No network access, no analytics, no tracking.

Installation

Get started in seconds.

1

Download

Grab the latest .zip from GitHub Releases

2

Unzip

Extract copippe.app from the archive

3

Move to Apps

Drag to your Applications folder and launch

Note: This app is not notarized.

macOS will block the app on first launch. Use one of the following to open it:

Option A — Run in Terminal:

xattr -cr /Applications/copippe.app

Option B — System Settings:

  1. Try to open copippe.app (it will be blocked)
  2. Open System Settings → Privacy & Security
  3. Scroll down and click Open Anyway
Requires macOS 14 (Sonoma) or later
