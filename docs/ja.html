copippe

macOS用の軽量クリップボードツール。書式除去と履歴管理をメニューバーから。

ダウンロード GitHub

English · 日本語

特徴

  • コピー時にリッチテキストの書式を自動除去 — 常にプレーンテキストで貼り付け
  • メニューバーから最大20件のクリップボード履歴にアクセス
  • メニューバーに常駐、Dockアイコンなし
  • アプリ再起動後も履歴を保持
  • ログイン時に自動起動
  • Swift & SwiftUIで構築 — 軽量なメモリフットプリント

インストール

GitHub Releases からダウンロードし、解凍後アプリケーションフォルダに移動して起動。

macOS 14 (Sonoma) 以降が必要です。

このアプリは公証（Notarization）されていません。初回起動時にmacOSによってブロックされます。

ターミナルで実行: xattr -cr /Applications/copippe.app

または: アプリを開こうとする → システム設定 → プライバシーとセキュリティ → このまま開く
