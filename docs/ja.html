copippe

macOS用のシンプルで軽量なクリップボードツール。書式の除去と履歴管理をメニューバーから。

ダウンロード GitHub

特徴

✂️

プレーンテキストペースト

コピー時にリッチテキストの書式（HTML、RTF、スタイル）を自動除去。常にクリーンなプレーンテキストで貼り付け。

📜

クリップボード履歴

メニューバーから最大20件の履歴にアクセス。クリックしてコピー、⌘Vで貼り付け。

🪶

軽量設計

SwiftとSwiftUIで構築。メモリ使用量を最小限に抑え、Dockにアイコンを表示せず、邪魔になりません。

💾

履歴の永続化

クリップボード履歴はローカルに保存され、アプリ再起動後も復元されます。

🚀

自動起動

macOSのログイン項目で自動起動。一度設定すれば意識する必要はありません。

🔒

プライバシー重視

すべてのデータはMac内に保存。ネットワーク通信、アナリティクス、トラッキングは一切ありません。

インストール

数秒で始められます。

1

ダウンロード

GitHub Releasesから最新の.zipを取得

2

解凍

アーカイブからcopippe.appを展開

3

アプリケーションに移動

アプリケーションフォルダにドラッグして起動

macOS 14 (Sonoma) 以降が必要です
