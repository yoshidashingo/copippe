# AGENTS.md

このファイルは、AI コーディングエージェント(Claude Code、Codex など)がこのリポジトリで作業する際のガイダンスを提供します。

## プロジェクト概要

macOS用の軽量クリップボードツール。メニューバーに常駐し、プレーンテキスト貼り付け・クリップボード履歴・画像サポートを提供する。

## 技術スタック

- Swift 6
- macOS 14+

## 開発コマンド

Xcode でビルド・実行する。

## 言語

日本語で応答してください。

## 作業ルール

- アーティファクト（ドキュメント、設定、生成物、画像、コード成果物など）に変更を加えた場合は、完了前に Red Team レビューを実施する。
- レビューで指摘や改善提案が残る場合は、提案を反映し、再レビューする。このレビュー → 改善 → 検証のループは収束するまで継続し、上限は最大10回とする。
- 収束条件は、重大または中程度の指摘がなく、残る指摘が採用不要または明示的に保留できる状態であること。
- 10回に達しても収束しない場合は、ブロッカーとして報告し、勝手にコミットやプッシュを行わない。
- 収束後、必要な検証と Git 状態確認を実施し、対象変更のみをコミットしてリモートへプッシュする。未関係のユーザー変更や未追跡ファイルは含めない。
